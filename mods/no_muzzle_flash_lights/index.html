No Muzzle Flash Lights

26.09.2021 :: c6 :: 2.0 #fnv  #fo3 

A simple mod that improves game performance in combat by disabling lights cast on the environment by muzzle flashes, which are very performance heavy in New Vegas and Fallout 3.

The mod can be anywhere in your load order.

New Vegas version does not take up a slot in the load order and comes with an .ini file in config/NMFL folder. By default, it only disables the lights for NPCs and leaves them on for the player.

Download links:

  • New Vegas (requires JohnnyGuitar NVSE 4.10+, JIP LN NVSE)
  • Fallout 3 (requires Command Extender)
