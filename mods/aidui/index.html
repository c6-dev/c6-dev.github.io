AidUI

30.09.2021 :: c6 :: 2.0 #fnv 

Active Ingestibles Duration UI or AidUI adds widgets on HUD for consumed Aid items that display the time until their effects run out.


Includes MCM options for changing size of widgets, their position, and Bars mode.

Requirements:⌗

  • xNVSE﻿
  • UIO
  • JIP LN NVSE﻿
  • JohnnyGuitar NVSE﻿
  • MCM

Known issues:⌗

  • Some icons are off-centered - the mod uses vanilla inventory icons, i can’t prevent it on my side. Bars mode does not have this problem.
  • After drinking several different alcoholic beverages only one icon is displayed - the game treats them as a single item if their effects are identical.

Update 2.0 30.09.2021:⌗

  • A clean save is recommended for this update.
  • Rewrote the entire mod using latest xNVSE features
  • Ditched .ini config in favor of a redesigned MCM menu
  • Added scale option
  • Added Bars mode
